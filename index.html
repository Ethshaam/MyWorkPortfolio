Cloud Computing

This is going to cover different aspects of cloud computing, ranging from the advantages and core principles of cloud computing to the amazon services and what the services provide

The cloud is:

Highly available meaning it's accessible when you need it. It's an on-demand service

Fault tolerance meaning it can withstand a certain amount of failure and remain functional. AWS uses Load balancers to enable this. A load balancer is when the program diverts traffic to another port so one port doesn't become too congested with information

Scalabity = ability to grow in size when required, meaning you make more servers if the demand requires it

Elasticity = Ability to be upsize and downsize when needed
